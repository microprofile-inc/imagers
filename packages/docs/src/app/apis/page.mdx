# Imagers APIs

`imagers` 通过Uint8Array交换数据, 处理后默认返回png, 可以使用 image_convert_format 转换格式。

```typescript
import {init, crop} from "@microprofile/imagers";
await init();

fetch('https://example.com/some.jpg').then(res => res.bytes()).then(bytes => {
    let newImage = crop(bytes, 0, 0, 100, 100);
    // new Blob([newImage], {type: 'image/png'});
    // new File([newImage], 'image.png');
    // some code...
})
```

# crop image
```typescript
fetch(png.src).then(r => r.bytes()).then(bytes => {
  init().then(() => {
    let result = crop(bytes, 1000, 1000, 100, 100);
    const blob = new Blob([result], {type: 'image/png'});
    setSrc(URL.createObjectURL( blob ));
  });
})
```
